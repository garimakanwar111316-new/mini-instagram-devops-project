# 🚀 Mini Instagram - DevOps Project

A photo-sharing web application built with Flask and deployed using Docker, Kubernetes, and CI/CD automation.

---

## 🌟 Features
- Upload and view images
- Containerized using Docker
- Deployed on Kubernetes (NodePort)
- CI/CD pipeline using GitHub Actions

---

## 🛠️ Tech Stack
- Python (Flask)
- Docker
- Kubernetes
- GitHub Actions

---

## ⚙️ Architecture

User → Flask App → Docker → Kubernetes → CI/CD Pipeline

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
